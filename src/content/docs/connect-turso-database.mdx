# Drizzle \<\> Turso Database

<Prerequisites>
- Drizzle을 사용한 데이터베이스 [연결 기본사항](/docs/connect-overview)
- Turso Database - [웹사이트](https://docs.turso.tech/introduction)
- Turso Database 드라이버 - [웹사이트](https://docs.turso.tech/connect/javascript) & [GitHub](https://github.com/tursodatabase/turso/tree/main/bindings/javascript)
</Prerequisites>

**[공식 웹사이트](https://docs.turso.tech/introduction)**에 따르면,
Turso는 AI 시대에 여러분의 큰 꿈을 실현할 수 있는 작은 데이터베이스입니다.

#### Step 1 - 패키지 설치
<Npm>
drizzle-orm@beta @tursodatabase/database
-D drizzle-kit@beta
</Npm>

#### Step 2 - 드라이버 초기화 및 쿼리 실행
```typescript
import { drizzle } from 'drizzle-orm/tursodatabase/database';

const db = drizzle('sqlite.db');

const result = await db.execute('select 1');
```

기존 드라이버를 제공해야 하는 경우:

```typescript
import { Database } from '@tursodatabase/drivers';
import { drizzle } from 'drizzle-orm/tursodatabase/database';

const client = new Database('sqlite.db');
const db = drizzle({ client });

const result = await db.execute('select 1');
```
