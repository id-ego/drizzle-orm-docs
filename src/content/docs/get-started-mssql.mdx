# Drizzle \<\> MSSQL

<Callout type='error'>
이 페이지는 Drizzle 버전 `1.0.0-beta.2` 이상에서 사용 가능한 개념을 설명합니다.
</Callout>

<br/>

<Prerequisites>
- Drizzle을 사용한 데이터베이스 [연결 기초](/docs/connect-overview)
- node-mssql [기초](https://github.com/tediousjs/node-mssql)
</Prerequisites>

Drizzle은 `mssql` 드라이버를 사용한 MSSQL 연결을 네이티브로 지원합니다.

#### Step 1 - 패키지 설치
<Npm>
drizzle-orm@beta mssql
-D drizzle-kit@beta
</Npm>

#### Step 2 - 드라이버 초기화 및 쿼리 실행
<CodeTabs items={["mssql", "mssql with config"]}>
```typescript copy
// Make sure to install the 'mssql' package
import { drizzle } from 'drizzle-orm/node-mssql';

const db = drizzle(process.env.DATABASE_URL);

const result = await db.execute('select 1');
```
```typescript copy
// Make sure to install the 'pg' package
import { drizzle } from 'drizzle-orm/node-mssql';

// You can specify any property from the mssql connection options
const db = drizzle({
  connection: {
    connectionString: process.env.DATABASE_URL,
    ssl: true
  }
});

const result = await db.execute('select 1');
```
</CodeTabs>

<Callout type='warning'>
`node-mssql` 드라이버는 `Pool` 초기화 시 `await`를 요구하기 때문에, 각 요청 전에 `await`를 사용해야 합니다. 단, Drizzle에 자체 Pool 인스턴스를 제공하는 경우는 예외입니다. 이 경우 `db.$client`에 접근하려면 먼저 `await`를 사용한 후 사용해야 합니다.

```ts
const awaitedClient = await db.$client;
const response = awaitedClient.query...
```
</Callout>

기존 드라이버를 제공해야 하는 경우:

```typescript copy
// Make sure to install the 'mssql' package
import { drizzle } from "drizzle-orm/node-mssql";
import type { ConnectionPool } from 'mssql';

const pool = await mssql.connect(connectionString);
const db = drizzle({ client: pool });

const result = await db.execute('select 1');
```
