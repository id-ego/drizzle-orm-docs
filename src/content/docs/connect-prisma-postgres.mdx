# Drizzle \<\> Prisma Postgres
<Prerequisites>
- Drizzle 데이터베이스 [연결 기초](/docs/connect-overview)
- Prisma Postgres 서버리스 데이터베이스 - [웹사이트](https://prisma.io/postgres)
- Prisma Postgres 직접 연결 - [문서](https://www.prisma.io/docs/postgres/database/direct-connections)
- Drizzle PostgreSQL 드라이버 - [문서](/docs/get-started-postgresql)
</Prerequisites>

Prisma Postgres는 [unikernels](https://www.prisma.io/blog/announcing-prisma-postgres-early-access) 기반으로 구축된 서버리스 데이터베이스입니다. 넉넉한 무료 티어, [작업 기반 요금제](https://www.prisma.io/blog/operations-based-billing), 그리고 콜드 스타트가 없는 것이 특징입니다.

PostgreSQL용 [`node-postgres`](https://node-postgres.com/) 또는 [`postgres.js`](https://github.com/porsager/postgres) 드라이버를 사용하여 연결할 수 있습니다.

<Callout type="info">
Prisma Postgres는 [서버리스 드라이버](https://www.prisma.io/docs/postgres/database/serverless-driver)도 제공하며, 향후 Drizzle ORM에서 지원될 예정입니다.
</Callout>

#### Step 2 - 드라이버 초기화 및 쿼리 실행
<CodeTabs items={["node-postgres (pg)", "postgres.js"]}>
```typescript 
// Make sure to install the 'pg' package 
import { drizzle } from "drizzle-orm/node-postgres";
import { Pool } from "pg";

const pool = new Pool({
  connectionString: process.env.DATABASE_URL,
});
const db = drizzle({ client: pool });
 
const result = await db.execute('select 1');
```
```typescript
// Make sure to install the 'postgres' package
import { drizzle } from 'drizzle-orm/postgres-js';
import postgres from 'postgres';

const queryClient = postgres(process.env.DATABASE_URL);
const db = drizzle({ client: queryClient });

const result = await db.execute('select 1');
```
