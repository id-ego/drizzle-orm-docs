## 커뮤니티 기여 릴리스

### 내부 기능 및 변경사항
- d1 테스트 세트 추가
- 내부 문서 이슈 수정

### 수정사항
- MySQL 타임스탬프 밀리초 잘림 문제 해결
- sqlite 기반 dialect의 `.get()` 메서드 타입 수정 ([#565](https://github.com/drizzle-team/drizzle-orm/issues/565))
- 쿼리가 두 번 실행되는 sqlite-proxy 버그 수정

### 새로운 패키지

[drizzle-typebox](/docs/typebox) 패키지에서 [Typebox](https://github.com/sinclairzx81/typebox) 지원이 추가되었습니다.

더 많은 사용 예제는 문서 페이지를 확인하세요: /docs/typebox
