# 기존 프로젝트에서 Drizzle과 CockroachDB 시작하기

<Callout type='error'>
이 페이지는 drizzle 버전 `1.0.0-beta.2` 이상에서 사용 가능한 개념을 설명합니다.
</Callout>

<br/>

<Prerequisites>
  - **dotenv** - 환경 변수 관리 패키지 - [여기서 읽기](https://www.npmjs.com/package/dotenv)
  - **tsx** - TypeScript 파일 실행 패키지 - [여기서 읽기](https://tsx.is/)
  - **node-postgres** - PostgreSQL 데이터베이스 쿼리 패키지 - [여기서 읽기](https://node-postgres.com/)
</Prerequisites>

<FileStructure/>

#### Step 1 - **node-postgres** 패키지 설치
<Npm>
  drizzle-orm@beta pg dotenv
  -D drizzle-kit@beta tsx @types/pg
</Npm>


#### Step 2 - 연결 변수 설정

<SetupEnv env_variable='DATABASE_URL' />

<Callout title='팁'>
PostgreSQL 데이터베이스가 아직 없고 테스트용으로 생성하려면, Docker에서 PostgreSQL을 설정하는 방법에 대한 가이드를 사용할 수 있습니다.

PostgreSQL in Docker 가이드는 [여기](/docs/guides/postgresql-local-setup)에서 확인할 수 있습니다. 가이드를 따라 설정하고, 데이터베이스 URL을 생성한 후(가이드에 설명되어 있음) 다음 단계로 돌아오세요.
</Callout>
