# Drizzle과 Bun:SQLite 시작하기

<Prerequisites>
  - **bun** - JavaScript 올인원 툴킷 - [자세히 보기](https://bun.sh/)
  - **Bun SQL** - PostgreSQL 데이터베이스 작업을 위한 네이티브 바인딩 - [자세히 보기](https://bun.sh/docs/api/sql)
</Prerequisites>

<Callout type='error'>
버전 `1.2.0`에서 Bun은 동시 문 실행에 문제가 있어, 여러 쿼리를 동시에 실행하려고 하면 오류가 발생할 수 있습니다.
이 문제를 추적할 수 있는 [GitHub 이슈](https://github.com/oven-sh/bun/issues/16774)를 생성했습니다. 이 문제가 해결되면 Bun SQL 측면에서 더 이상 이러한 오류가 발생하지 않습니다.
</Callout>
