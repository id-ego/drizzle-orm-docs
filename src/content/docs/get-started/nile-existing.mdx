#### Step 4 - 데이터베이스 인트로스펙션

Drizzle Kit은 데이터베이스를 인트로스펙트하고 마이그레이션과 함께 스키마 파일을 생성하는 CLI 명령어를 제공합니다. 스키마 파일에는 데이터베이스 테이블, 컬럼, 관계 및 인덱스에 대한 모든 정보가 포함됩니다.

예를 들어, 데이터베이스에 다음과 같은 테이블이 있다고 가정해 봅시다:

```sql copy
CREATE TABLE IF NOT EXISTS "todos" (
  "id" uuid DEFAULT gen_random_uuid(),
  "tenant_id" uuid,
  "title" varchar(256),
  "estimate" varchar(256),
  "embedding" vector(3),
  "complete" boolean
);
```

데이터베이스 스키마를 가져옵니다:

```bash copy
npx drizzle-kit pull
```

인트로스펙션 결과로 `schema.ts` 파일, 데이터베이스 스키마 스냅샷이 포함된 `meta` 폴더, 마이그레이션 sql 파일, 그리고 [관계형 쿼리](/docs/rqb)를 위한 `relations.ts` 파일이 생성됩니다.

<Callout title='내장 테이블'>
Nile은 모든 데이터베이스에 포함된 여러 내장 테이블을 가지고 있습니다. Nile 데이터베이스를 인트로스펙트하면 내장 테이블이 포함됩니다.
예를 들어 아래 예제에서 볼 수 있는 `tenants` 테이블이 있습니다. 이를 통해 새 테넌트 생성, 테넌트 목록 조회 및 기타 작업을 쉽게 수행할 수 있습니다.
</Callout>

다음은 생성된 `schema.ts` 파일의 예제입니다:

```typescript copy filename="src/db/schema.ts"
// table schema generated by introspection
import { pgTable, uuid, text, timestamp, varchar, vector, boolean } from "drizzle-orm/pg-core"
import { sql } from "drizzle-orm"

export const tenants = pgTable("tenants", {
	id: uuid().default(sql`public.uuid_generate_v7()`).primaryKey().notNull(),
	name: text(),
	created: timestamp({ mode: 'string' }).default(sql`LOCALTIMESTAMP`).notNull(),
	updated: timestamp({ mode: 'string' }).default(sql`LOCALTIMESTAMP`).notNull(),
	deleted: timestamp({ mode: 'string' }),
});

export const todos = pgTable("todos", {
	id: uuid().defaultRandom(),
	tenantId: uuid("tenant_id"),
	title: varchar({ length: 256 }),
	estimate: varchar({ length: 256 }),
	embedding: vector({ dimensions: 3 }),
	complete: boolean(),
});
```

인트로스펙션에 대한 자세한 내용은 [문서](/docs/drizzle-kit-pull)를 참조하세요.

#### Step 9 - 테이블 스키마 업데이트 (선택 사항)

테이블 스키마를 업데이트하려면 `schema.ts` 파일에서 수정할 수 있습니다. 예를 들어, `todos` 테이블에 새로운 `deadline` 컬럼을 추가해 봅시다:

```typescript copy filename="src/db/schema.ts" {19}
import { pgTable, uuid, text, timestamp, varchar, vector, boolean } from "drizzle-orm/pg-core"
import { sql } from "drizzle-orm"

export const tenants = pgTable("tenants", {
	id: uuid().default(sql`public.uuid_generate_v7()`).primaryKey().notNull(),
	name: text(),
	created: timestamp({ mode: 'string' }).default(sql`LOCALTIMESTAMP`).notNull(),
	updated: timestamp({ mode: 'string' }).default(sql`LOCALTIMESTAMP`).notNull(),
	deleted: timestamp({ mode: 'string' }),
});

export const todos = pgTable("todos", {
	id: uuid().defaultRandom(),
	tenantId: uuid("tenant_id"),
	title: varchar({ length: 256 }),
	estimate: varchar({ length: 256 }),
	embedding: vector({ dimensions: 3 }),
	complete: boolean(),
  deadline: timestamp({ mode: 'string' })
});
```
