# Drizzle과 Bun:SQLite 시작하기

<Prerequisites>
  - **bun** - JavaScript 올인원 툴킷 - [자세히 보기](https://bun.sh/)
  - **bun:sqlite** - 고성능 SQLite3 드라이버의 네이티브 구현 - [자세히 보기](https://bun.sh/docs/api/sqlite)
</Prerequisites>

<FileStructure />

#### Step 1 - 필수 패키지 설치
<Npm>
  drizzle-orm
  -D drizzle-kit @types/bun
</Npm>

#### Step 2 - 연결 변수 설정

<SetupEnv env_variable='DB_FILE_NAME' />

<Callout type='info' title='중요'>
예를 들어, 테스트 목적으로 프로젝트 루트에 SQLite 데이터베이스 파일을 생성하려면 다음 예제를 사용할 수 있습니다:
```plaintext copy
DB_FILE_NAME=mydb.sqlite
```
</Callout>

`bun`으로 스크립트를 실행하려면 다음 명령어를 사용합니다:
```bash copy
bun src/index.ts
```
