# Drizzle과 CockroachDB 시작하기

<Callout type='error'>
이 페이지는 Drizzle 버전 `1.0.0-beta.2` 이상에서 사용 가능한 개념을 설명합니다.
</Callout>

<br/>

<Prerequisites>
  - **dotenv** - 환경 변수 관리 패키지 - [자세히 보기](https://www.npmjs.com/package/dotenv)
  - **tsx** - TypeScript 파일 실행 패키지 - [자세히 보기](https://tsx.is/)
  - **node-postgres** - PostgreSQL 데이터베이스 쿼리 패키지 - [자세히 보기](https://node-postgres.com/)
</Prerequisites>

Drizzle은 `node-postgres`와 `postgres.js` 드라이버를 통한 CockroachDB 연결을 기본적으로 지원합니다.

이 시작 가이드에서는 `node-postgres`를 사용합니다. PostgreSQL에 연결하는 다양한 방법을 알아보려면 [CockroachDB 연결](/docs/get-started-cockroach) 페이지를 확인하세요.
