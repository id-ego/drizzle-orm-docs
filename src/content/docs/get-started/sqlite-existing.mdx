<Callout type='info' title='중요'>
예를 들어, 테스트 목적으로 프로젝트 루트에 SQLite 데이터베이스 파일을 생성하려면 `LibSQL`에서 요구하는 형식에 따라 실제 파일명 앞에 `file:`을 사용해야 합니다:
```plaintext copy
DB_FILE_NAME=file:local.db
```
자세한 정보는 **[LibSQL 문서](https://docs.turso.tech/sdk/ts/reference#local-development)**를 확인하세요.
</Callout>
