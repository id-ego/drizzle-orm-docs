# 기존 프로젝트에서 Drizzle과 MSSQL 시작하기

<Callout type='error'>
이 페이지는 drizzle `1.0.0-beta.2` 이상 버전에서 사용 가능한 개념을 설명합니다.
</Callout>

<br/>

<Prerequisites>
  - **dotenv** - 환경 변수 관리 패키지 - [여기서 읽기](https://www.npmjs.com/package/dotenv)
  - **tsx** - TypeScript 파일 실행 패키지 - [여기서 읽기](https://tsx.is/)
  - **node-mssql** - MSSQL 데이터베이스 쿼리 패키지 - [여기서 읽기](https://github.com/tediousjs/node-mssql)
</Prerequisites>
