# Drizzle과 PostgreSQL 시작하기

<Prerequisites>
  - **dotenv** - 환경 변수 관리 패키지 - [자세히 보기](https://www.npmjs.com/package/dotenv)
  - **tsx** - TypeScript 파일 실행 패키지 - [자세히 보기](https://tsx.is/)
  - **node-postgres** - PostgreSQL 데이터베이스 쿼리 패키지 - [자세히 보기](https://node-postgres.com/)
</Prerequisites>

Drizzle은 `node-postgres`와 `postgres.js` 드라이버를 통한 PostgreSQL 연결을 기본적으로 지원합니다.

이 시작 가이드에서는 `node-postgres`를 사용합니다. PostgreSQL에 연결하는 다양한 방법을 확인하려면 [PostgreSQL 연결](/docs/get-started-postgresql) 페이지를 참조하세요.

<Callout title='팁'>
아직 PostgreSQL 데이터베이스가 없고 테스트용으로 생성하고 싶다면, Docker에서 PostgreSQL을 설정하는 가이드를 참조하세요.

Docker에서 PostgreSQL 설정 가이드는 [여기](/docs/guides/postgresql-local-setup)에서 확인할 수 있습니다. 가이드를 따라 설정하고 데이터베이스 URL을 생성한 후(가이드에 설명되어 있음) 다음 단계로 돌아오세요.
</Callout>
