# Drizzle과 SQLite 시작하기

<Prerequisites>
  - **dotenv** - 환경 변수 관리 패키지 - [자세히 보기](https://www.npmjs.com/package/dotenv)
  - **tsx** - TypeScript 파일 실행 패키지 - [자세히 보기](https://tsx.is/)
  - **libsql** - 낮은 쿼리 지연 시간에 최적화된 SQLite 포크로, 글로벌 애플리케이션에 적합합니다 - [자세히 보기](https://docs.turso.tech/libsql)
</Prerequisites>

Drizzle은 `libsql`과 `better-sqlite3` 드라이버로 SQLite 연결을 기본 지원합니다.

이 시작 가이드에서는 `libsql`을 사용합니다. SQLite 연결 방법을 더 찾고 싶다면
[SQLite 연결](/docs/get-started-sqlite) 페이지를 확인하세요.

<Callout type='info' title='중요'>
예를 들어, 테스트 목적으로 프로젝트 루트에 SQLite 데이터베이스 파일을 생성하려면 실제 파일명 앞에 `file:`을 사용해야 합니다. 이는 `LibSQL`에서 요구하는 형식입니다:
```plaintext copy
DB_FILE_NAME=file:local.db
```
자세한 내용은 **[LibSQL 문서](https://docs.turso.tech/sdk/ts/reference#local-development)**를 확인하세요.
</Callout>
