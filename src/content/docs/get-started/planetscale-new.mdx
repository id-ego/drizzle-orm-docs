# Drizzle과 PlanetScale 시작하기

<Prerequisites>
  - **dotenv** - 환경 변수 관리 패키지 - [여기에서 읽기](https://www.npmjs.com/package/dotenv)
  - **tsx** - TypeScript 파일 실행 패키지 - [여기에서 읽기](https://tsx.is)
  - **PlanetScale** - MySQL 데이터베이스 플랫폼 - [여기에서 읽기](https://planetscale.com)
  - **database-js** - PlanetScale 서버리스 드라이버 - [여기에서 읽기](https://github.com/planetscale/database-js)
</Prerequisites>

<Callout title='PlanetScale also offers Postgres' type='info'>
Looking for PostgreSQL? Check out our [PlanetScale Postgres guide](/docs/get-started/planetscale-postgres-new)
</Callout>

<Callout title='important' type='warning'>
이 튜토리얼에서는 `database-js` 드라이버를 사용하여 PlanetScale 데이터베이스에 **HTTP** 호출을 합니다. TCP를 통해 PlanetScale에 연결해야 하는 경우 [MySQL 시작하기](/docs/get-started/mysql-new) 페이지를 참조하세요.
</Callout>

#### Step 1 - **@planetscale/database** 패키지 설치

<InstallPackages lib='@planetscale/database'/>

#### Step 2 - 연결 변수 설정

프로젝트 루트에 `.env` 파일을 생성하고 데이터베이스 연결 변수를 추가합니다:

```plaintext copy
DATABASE_HOST=
DATABASE_USERNAME=
DATABASE_PASSWORD=
```

<Callout title='tips'>
`database-js` 드라이버를 통해 연결하는 데 필요한 모든 환경 변수를 얻으려면 [PlanetScale 문서](https://planetscale.com/docs/tutorials/planetscale-serverless-driver-node-example#use-the-sample-repository)를 확인하세요.
</Callout>
