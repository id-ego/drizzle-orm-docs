# 기존 프로젝트에서 Drizzle과 SQLite 시작하기

<Prerequisites>
  - **dotenv** - 환경 변수 관리 패키지 - [여기에서 읽기](https://www.npmjs.com/package/dotenv)
  - **bun** - JavaScript 올인원 툴킷 - [여기에서 읽기](https://bun.sh/)
  - **Bun SQL** - PostgreSQL 데이터베이스 작업을 위한 네이티브 바인딩 - [여기에서 읽기](https://bun.sh/docs/api/sql)
</Prerequisites>

<Callout type='error'>
버전 `1.2.0`에서 Bun은 동시 문(concurrent statements) 실행에 문제가 있으며, 여러 쿼리를 동시에 실행하려고 하면 오류가 발생할 수 있습니다.
이에 대한 [GitHub 이슈](https://github.com/oven-sh/bun/issues/16774)를 생성했습니다. 수정되면 Bun의 SQL 측에서 더 이상 이러한 오류가 발생하지 않습니다.
</Callout>
