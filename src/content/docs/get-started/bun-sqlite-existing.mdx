# 기존 프로젝트에서 Drizzle과 Bun:SQLite 시작하기

<Prerequisites>
  - **dotenv** - 환경 변수 관리 패키지 - [자세히 보기](https://www.npmjs.com/package/dotenv)
  - **tsx** - TypeScript 파일 실행 패키지 - [자세히 보기](https://tsx.is/)
  - **bun** - JavaScript 올인원 툴킷 - [자세히 보기](https://bun.sh/)
  - **bun:sqlite** - 고성능 SQLite3 드라이버의 네이티브 구현 - [자세히 보기](https://bun.sh/docs/api/sqlite)
</Prerequisites>

</Npm>

#### Step 2 - 연결 변수 설정

<SetupEnv env_variable='DB_FILE_NAME' />

<Callout type='info' title='중요'>
예를 들어, 프로젝트 루트에 SQLite 데이터베이스 파일이 있다면 다음 예시를 사용할 수 있습니다:
```plaintext copy
DB_FILE_NAME=mydb.sqlite
```
