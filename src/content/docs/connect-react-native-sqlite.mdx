# Drizzle \<\> React Native SQLite
React Native 앱에서 Drizzle ORM을 사용하려면 [`Expo SQLite`](#expo-sqlite)를 사용하세요.
저희가 찾은 유일한 [인기 라이브러리](https://github.com/andpor/react-native-sqlite-storage)는 새로운 Hermes JavaScript 런타임을 지원하지 않습니다.
Hermes는 현재 React Native와 Expo의 기본 런타임입니다.
